# Custom MCB Background Music
自定义我的世界基岩版背景音乐资源包

## Usage

### 系统要求：
- nodejs > 14
or
- powershell > 7

### 使用步骤

#### 1.准备music
准备音频文件，音频格式为`.ogg`
转换网站：https://convertio.co/zh/mp4-ogg/

#### 2.将ogg文件放入`/sounds/music/game/`目录下
如下目录：
![Alt text](image.png)

#### 3.执行powershell脚本或者js脚本

windows在包的根目录下执行powershell脚本：
> .\CBM.ps1

或者执行js脚本：
> node .\CBM.js
